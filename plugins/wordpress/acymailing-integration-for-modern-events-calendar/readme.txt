== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Modern Events Calendar](https://wordpress.org/plugins/modern-events-calendar-lite/)

Add events from Modern Events Calendar to your emails.
Filter users on events participation.

Features :

- Add a new droppable button on the editor to add events from Modern Events Calendar to your email
- Display events in the email
- Select the data you want to display (pictures, title, link etc)
- Customize the html render in the plugin settings
- Filter users on events participation


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Modern Events Calendar AcyMailing`
3. Find it, install it, enable it
4. You're done!
