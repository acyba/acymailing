Add AcyMailing lists to your Contact Form 7 forms

== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Contact Form 7](https://wordpress.org/plugins/contact-form-7/)

Add AcyMailing lists to your Contact Form 7 forms

Features :

- Add a new button on forms edition to add AcyMailing lists
- Display AcyMailing lists in the forms
- Create AcyMailing users and subscribe them to the lists they selected


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Contact Form 7 AcyMailing`
3. Find it, install it, enable it
4. You're done!
