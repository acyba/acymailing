== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Business Directory](https://wordpress.org/plugins/business-directory-plugin/)

Add listings from Business Directory to your emails.

Features :

- Add a new droppable button on the editor to add listings from Business Directory to your emails
- Display listings in the email
- Select the data you want to display (image, title, link etc)
- Customize the html render in the plugin settings


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Business Directory AcyMailing`
3. Find it, install it, enable it
4. You're done!
