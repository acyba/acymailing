== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)

Insert a dynamic table of contents in your emails based on their contents via the AcyMailing drag and drop editor

Features :

- Add a new droppable button on the editor to generate a table of contents in your email
- Select which elements of your emails should be listed (h1, h2, etc)
- Choose between two displays


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Table of contents AcyMailing`
3. Find it, install it, enable it
4. You're done!
