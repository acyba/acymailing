Export your AcyMailing users through the automations

== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)

Export your AcyMailing users automatically via the automation.

Features :

- Add an action in the automation to export your users
- Export every data of your users
- Choose the encoding
- Choose the path and the name of the file where you export your users


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Export AcyMailing users`
3. Find it, install it, enable it
4. You're done!
