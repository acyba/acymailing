Add AcyMailing lists to your Ultimate Member forms

== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Ultimate Member](https://fr.wordpress.org/plugins/ultimate-member/)

Add AcyMailing lists to your Ultimate Member forms

Features :

- Add a new section in the AcyMailing configuration to add lists to your forms
- Display AcyMailing lists in the forms
- Create AcyMailing users and subscribe them to the lists they selected


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Ultimate Member AcyMailing`
3. Find it, install it, enable it
4. You're done!
