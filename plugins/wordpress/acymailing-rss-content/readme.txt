Add RSS feed to your emails via the AcyMailing drag and drop editor

== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)

Add RSS feed to your emails via the AcyMailing drag and drop editor

Features :

- Add a new droppable button on the editor to add RSS feed to your email
- Display all article in the email
- Select the data you want to display (pictures, title, link etc)
- Customize the html render in the plugin settings


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `RSS AcyMailing`
3. Find it, install it, enable it
4. You're done!
