Trigger AcyMailing automation when a user subscribe to your membership, filter users on their subscription, add custom value to your email

== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [MemberPress](https://memberpress.com/)

Features :

- Add a new trigger in the automation when a user subscribe to a new membership
- Filter your users on their member subscription
- Add users custom field value to your emails


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `MemberPress AcyMailing`
3. Find it, install it, enable it
4. You're done!
