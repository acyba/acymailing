== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)

Add custom email headers to the sent emails.


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Custom headers AcyMailing`
3. Find it, install it, enable it
4. You're done!
