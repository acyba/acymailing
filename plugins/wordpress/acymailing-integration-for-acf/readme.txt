== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/)

Insert custom posts from Advanced Custom Fields in your emails.
Automatically generate monthly newsletters including the last posts.

Features :

- Adds a new droppable button in the editor to insert custom posts in your emails
- Display custom posts in the email effortlessly
- Select the data you want to display (pictures, title, description, link, etcâ€¦)
- Customize the way the posts are shown in the plugin settings


== Installation ==

1. In your Admin click on Plugins then "Add New Plugin"
2. Search for `Advanced Custom Fields AcyMailing`
3. Find it, install it, enable it
4. You're done!
