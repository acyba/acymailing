Filter AcyMailing users on their Learndash groups and attended courses

== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Learndash](https://www.learndash.com/)

Features :

- Filter your users on their Learndash groups
- Filter your users on attended courses


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Learndash AcyMailing`
3. Find it, install it, enable it
4. You're done!
