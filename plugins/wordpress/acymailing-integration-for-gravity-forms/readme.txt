Add AcyMailing lists to your Gravity Forms forms

== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Gravity Forms](https://www.gravityforms.com/)

Add AcyMailing lists to your Gravity Forms forms

Features :

- Add a new droppable button on forms edition to add AcyMailing lists
- Display AcyMailing lists in the forms
- Create AcyMailing users and subscribe them to the lists they selected


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Gravity Forms AcyMailing`
3. Find it, install it, enable it
4. You're done!
