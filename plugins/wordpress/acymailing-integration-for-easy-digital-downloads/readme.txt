== Description ==

> #### Requirements
> [AcyMailing](https://wordpress.org/plugins/acymailing/)
> [Easy Digital Downloads](https://wordpress.org/plugins/easy-digital-downloads/)

Add products from Easy Digital Downloads to your emails.
Create individual coupon codes for your receivers.

Features :

- Add a new droppable button on the editor to add products to your emails
- Display products in the email
- Select the data you want to display (pictures, title, link etc)
- Customize the html render in the plugin settings
- Automatically create individual coupon codes for each receiver


== Installation ==

1. In your Admin click on your Plugin then "Add New"
2. Search for `Easy Digital Downloads AcyMailing`
3. Find it, install it, enable it
4. You're done!
